# BOOTCAMP-PROD

A premium, high-visibility tech bootcamp platform built with React, Vite, and GSAP.

## Features
- **Luminous BentoGrid**: Premium module display with interactive particle effects and 3D tilt.
- **Expert-Led Modules**: Structured learning paths for Business Analysis, HR, and Web Development.
- **Real-Time Interactive**: Character-by-character SplitText animations and glassmorphism UI.
- **Full-Stack Integration**: Express/Node.js backend with attendance tracking and persistent storage.

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install` and `cd backend && npm install`.
3. Start the development server: `npm run dev`.
4. Start the backend: `cd backend && npm start`.
